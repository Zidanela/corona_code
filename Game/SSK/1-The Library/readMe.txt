Usage:

1. Copy the folder ssk/ into your project's root directory.

2. Add these lines of code to your main.lua file

require( "ssk.globals" ) 
require( "data.globals" )

3. Done...for now.  There are additional features you can enable (with a few lines of code).


For more information about SSK and its features, please head over to my site:
