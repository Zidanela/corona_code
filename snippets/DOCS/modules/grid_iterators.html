GridIter (c1, r1, c2, r2, dw, dh, ncols)
Iterator over a rectangular region on an array-based grid.

Parameters:

  • c1: uint Column index #1.
  • r1: uint Row index #1.
  • c2: uint Column index #2.
  • r2: uint Row index #2.
  • dw: number Uniform cell width.
  • dh: number Uniform cell height.
  • ncols: uint Number of columns in a grid row. If absent, this is assumed to be the larger of c1 and c2.

Returns:

    iterator

    Supplies the following, in order, at each iteration:

    • Current iteration index.
    • Array index, as per index_ops.CellToIndex.
    • Column index.
    • Row index.
    • Cell corner x-coordinate, 0 at c = 1.
    • Cell corner y-coordinate, 0 at r = 1.

see also:

    iterators.InstancedAutocacher
