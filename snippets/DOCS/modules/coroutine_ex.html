Module coroutine_ex

An extended coroutine wrapper behaves like a function returned by coroutine.wrap, though as a loop and not a one-shot call.

Once the body has completed, it will "rewind" and thus be back in its original state, excepting any side effects. It is also possible to query if the body has just rewound.

In addition, a coroutine created with this function can be reset, i.e. the body function is explicitly rewound while active. To accommodate this, reset logic can be attached to clean up any important state.
