Module effect.Stash

Pools for various resources, to be recycled through effects.

Functions

PullCircle (what, into, new)
If a circle has been stashed in a certain cache, it will be pulled out and recycled. Otherwise, a fresh circle is created with default properties.

Parameters:

  • what: Name of a cache which may contain circles deposited by PushCircle.
  • into: DisplayGroup Group into which the circle is pulled. If absent, the stage.
  • new: callable Optional circle constructor, which takes into as argument and returns a circle display object; if absent, a default is used.

Returns:

    DisplayCircle Circle display object.
PullRect (what, into, new)
If a rect has been stashed in a certain cache, it will be pulled out and recycled. Otherwise, a fresh rect is created with default properties.

Parameters:

  • what: Name of a cache which may contain rects deposited by PushRect.
  • into: DisplayGroup Group into which the rect is pulled. If absent, the stage.
  • new: callable Optional rect constructor, which takes into as argument and returns a rect display object; if absent, a default is used.

Returns:

    DisplayRect Rect display object.
PushCircle (what, circle, how)
Stashes one or more circles for later retrieval by PullCircle.

Parameters:

  • what: Name of a cache (will be created if absent).
  • circle: DisplayObject A group or circle, pushed according to how.
  • how: string As per PushRect, mutatis mutandis.
PushRect (what, rect, how)
Stashes one or more rects for later retrieval by PullRect.

Parameters:

  • what: Name of a cache (will be created if absent).
  • rect: DisplayObject A group or rect, pushed according to how.
  • how: string If this is "is_group", rect must be a display group containing only rects. All of its elements will be emptied into the cache.

    If it is "is_dead_group", rect is also such a group. In this case, elements are gradually transferred to the cache, after which rect will remove itself.

    Otherwise, rect is a display rect and is sent to the cache.
