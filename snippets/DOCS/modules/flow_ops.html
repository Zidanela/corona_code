Wait (duration, update, arg, yvalue)
Waits for some time to pass.

Built on top of flow_bodies.Body_Timed.

Parameters:

  • duration: number Time to wait.
  • update: callable Optional update logic, called as

    update(time_state, duration, arg),

    with time_state as per flow_bodies.Body_Timed.

  • arg: Argument.
  • yvalue: Optional value to yield.

Returns:

    boolean The wait completed?
WaitForMultipleSignals (signals, count, pred, update, arg, yvalue)
Waits for a group of signals to reach a certain state.

Built on top of flow_bodies.Body.

Parameters:

  • signals: Callable or read-indexable signal object. For i = 1 to count, the corresponding test is performed: signals(i) or signals[i].

    A test passes if the return or lookup result is true.

  • count: uint Signal count.
  • pred: string

    Predicate name, which may be any of the following:

    • "all": All tests must pass.
    • "any": At least one test must pass.
    • "any_not: At least one test must not pass.
    • "none": No test may pass.
  • update: callable

    Optional update logic, called as

    update(signals, count, arg).
  • arg: Argument.
  • yvalue: Optional value to yield.

Returns:

    The signals satisfied the predicate?

see also:

  • array_ops.All
  • array_ops.Any
