dgroup:BindDefaults (defs)
Associates a set of default values to the dialog.

Parameters:

  • defs: table A list of default values, or nil to unbind defaults.

    The dialog retains this reference, and thus is aware of any external changes.

dgroup:BindValues (values)
Associates a set of values with the dialog.

Parameters:

  • values: table A list of values, or nil to unbind values.

    The dialog retains this reference, and thus is aware of any external changes. More importantly, these are the values modified by the dialog components.

dgroup:Find (name)
Searches by name for an object in the dialog.

Parameters:

  • name: Object name, as passed through name in the object's options. If the name was true, the final name will be the value of value_name.

Returns:

    DisplayObject Object, or nil if not found.
